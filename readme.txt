== Installation ==

<h3>From your WordPress dashboard</h3>

<ol>
<li>Visit ‘Plugins > Add New’</li>
<li>Search for ‘c4d woo carousel’</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
 

<h3>From WordPress.org</h3>

<ol>
<li>Download Woo Carousel.</li>
<li>Upload the ‘c4d-woo-carousel’ directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)</li>
<li>Activate plugin from your Plugins page.</li>
</ol>
